title: 'Integrations'
---

Loops currently supports:

- **[Framer](/forms/framer)**
  - Custom form component
- **[Webflow](/forms/native-webflow)**
  - Native Webflow forms with Loops via form endpoint
  - Custom Loops form embeds
- **[Zapier](/add-users/zapier)**
  - Connect to 1000's of tools
- **[Segment](add-users/segment)**
  - Import contacts
  - Events and contact properties coming soon
- **[Sales tools](/events/auto-bcc)**
  - Hubspot, Salesforce, etc via BCC
- **[Custom integration](/add-users/api-reference)**
  - Using our form endpoint or API you can connect Loops to most tools
- **[Emailify](/creating-emails/uploading-custom-email)**
  - Import a custom MJML email created in Emailify
